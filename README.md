# 🎸 ציידות השדים של ה-KPOP - אנגלית לכיתה ג'

אתר אינטראקטיבי ומשחקי ללמידת אנגלית עבור ילדי כיתה ג' דוברי עברית.

## 🎮 סיפור המשחק

הצטרפו לגיבורי ה-K-Pop ועזרו להם להילחם בשדים שגונבים מילים באנגלית! כל משחק מביס שד אחד ומקנה לכם כוכבים ותגים!

## 🎯 המשחקים

### 1. 🎯 תופס המילים (Word Catcher)
- **מטרה**: למידת אוצר מילים בסיסי
- **איך משחקים**: תפסו את התמונה הנכונה למילה
- **מילים**: cat, dog, apple, sun, star, car, book, ball
- **כישורים**: זיהוי מילים והתאמה לתמונות

### 2. ✨ איות הקסם (Spell the Magic)
- **מטרה**: למידת איות
- **איך משחקים**: סדרו את האותיות ליצירת המילה הנכונה
- **מילים**: cat, dog, sun, book, ball, tree
- **כישורים**: איות וזיהוי אותיות

### 3. 🎤 מדברים כדי לנצח (Speak to Defeat)
- **מטרה**: תרגול הגייה
- **איך משחקים**: תשמעו את המילה ותגידו אותה בקול
- **מילים**: hello, thank you, school, friend, happy, water
- **כישורים**: הגייה נכונה (משתמש ב-Web Speech API)

### 4. 💪 בחר את הכוח (Choose the Power)
- **מטרה**: הבנת קריאה ואוצר מילים
- **איך משחקים**: בחרו את התשובה הנכונה לשאלה
- **נושאים**: צבעים, בעלי חיים, פעולות יומיומיות
- **כישורים**: הבנת קריאה בסיסית

## 🌟 מערכת התקדמות

- ⭐ **כוכבים**: כל משחק שמושלם מעניק כוכבים
- 🏆 **תגים**: כל משחק שמושלם מעניק תג מיוחד
- 💾 **שמירת התקדמות**: ההתקדמות נשמרת אוטומטית ב-localStorage
- 🎉 **מסך ניצחון**: כשמסיימים את כל המשחקים מקבלים מסך ניצחון מיוחד

## 🚀 התחלת פיתוח

### דרישות מקדימות
- Node.js (גרסה 18 ומעלה)
- npm או yarn

### התקנה
```bash
npm install
```

### הרצה במצב פיתוח
```bash
npm run dev
```

האתר יעלה בכתובת: `http://localhost:5173`

### בנייה לפרודקשן
```bash
npm run build
```

### תצוגה מקדימה של build
```bash
npm run preview
```
